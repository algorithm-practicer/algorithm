### 문제 풀이

- 각 주차별로 `W{주차}-{년도}`의 폴더명을 갖는 폴더에 문제풀이 파일들을 업로드한다.
  - e.g., 2025년 3주차인 경우, W03-2025
- 파일명은 {Platform}-{문제번호}-{사용자이름}의 형태를 갖는다.
  - e.g., BOJ-1234-chulsu.md
- 파일 형태는 [해당 마크다운 템플릿](./docs/solved-problem.md)을 따르며, 문제의 성격에 따라 추가적인 내용은 상관 없다.
